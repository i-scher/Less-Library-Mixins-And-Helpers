Original CSS shapes taken directly from The Shapes of CSS (http://css-tricks.com/examples/ShapesOfCSS/)
Original CSS animations taken directly from CSS3 ANIMATION CHEAT SHEETS (www.justinaguilar.com/animations/)

This is a mixin library for LESS CSS. Import this mixin library to your LESS CSS files and you can create advanced shapes, shadows, gradients... with a single HTML element and simplified LESS code that creates very fancy CSS tricks. 

All of this work with at least one browser.
